### 参与社区文档

**投稿 | 翻译文章**

```bash
git clone https://github.com/linuxdeepin/docs.git
nmp install
编写 Marckdown 文章，放入合适文件夹
npm run docs:dev  # 本地预览效果，且会自动生成头部和永久链接
```

参考：[链接](https://docs.deepin.org/pages/d318e7/)
