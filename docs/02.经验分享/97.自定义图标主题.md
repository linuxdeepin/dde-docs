# 自定义图标主题

1. 前往 Gnome Look 站点（https://www.gnome-look.org/）下载自己喜欢的图标主题（Full Icon Themes分类下）。
2. 解压至 /usr/share/icons 或用户目录下的 .icons 文件夹下。
3. 在系统设置 - 个性化 - 图标主题中选择安装的图标。
